# MS1-Based-Workflow
MS1-Based Workflow for the Identification of Plastic Oligomers and Non-Intentionally Added Substances (NIAS)

This repository contains two independent workflows designed for compound identification. While they share a similar structural design, the key distinction lies in the use of internal standards — one workflow leverages standards, while the other does not.

To prevent naming conflicts between shared class names, the codebase is organized into two separate folders. Each folder contains a fully modular pipeline with its own classes and Jupyter notebook.
